Very simple plugin for inserting your Microsoft Dynamics Marketing tracking code on every page.

== Description ==

This is a bare bones option for people who are looking to simply insert the basic Microsoft Dynamics Marketing Web Site tracking code into every page without any fuss. 
I've only included the essential options; it's not very exciting, but works.

== Installation ==

1. Download the latest zip file and extract the plugin zip.
2. Upload the this directory inside your `/wp-content/plugins/` directory.
3. Activate plugin on the 'Plugins' menu in WordPress.
4. Click 'Settings' underneath the plugin or navigate to Settings > Dynamics Marketing Tracking and insert your tracking code and click the checkbox to turn it on.
5. That's it.
